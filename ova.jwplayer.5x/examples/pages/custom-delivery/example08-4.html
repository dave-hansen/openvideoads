Example - Selection of media based on bitrate - high, wrapped ad

This example illustrates how to specify a 'bitrate' as a selection criteria for the mediafiles returned by the openX ad server.


The configuration for this example is:


"" is defined as: 


<config>
   <ova.title>
       Example 8 - Selection of media based on bitrate
   </ova.title>
   <ova.json>
       {
            "ads": {
                "bitrate": "high",
                "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "../../../dist/templates/bitrated/vast1-wrapper-bitrates-tag.xml"
                  }
                ]
            },

            "debug": {
                "levels": "fatal, config, vast_template, vpaid, http_calls, playlist, api"
            }
       }
   </ova.json>
</config>
