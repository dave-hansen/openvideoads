OVA for JW Player 5 - Worked Examples

A number of examples have been included with this release to showcase the available range of configuration options. All of the examples require JW Player 5.8 or greater.

Broadly speaking, the examples have been organised into the following groupings:

  • Introductory Demo
    • VAST/VPAID Tag Inspector
  • Configuring JW Player 5 with OVA
    • Using the JW Embedder (preferred approach)
    • Using OBJECT/EMBED Tags
      • Setup via an external config file
      • Using "ova.tag" flashvar
      • Using "ova.json" flashvar
    • Using SWFObject
      • Via "ova.json"
    • Turning off OVA debug
    • JW plugins, playlists and general options
  • Supported VAST 1, 2 or 3 Ad Types
    • Basic Ad Formats
    • Delivering RTMP Linear Ads
    • Non-linear ads (Overlays)
    • Interactive (VPAID 1 or 2) Ads
    • Displaying Companions
    • Delivering "Ad Stories"
    • Ads on Live Streams
    • Deploying Surveys
  • Ad Tags, Calls and Servers
    • Delayed and On-Demand Ad Loading
    • Configuring the Ad Servers
    • Using OVA Variables and Transformers with Ad Tags
    • VAST Wrapper
    • Direct VAST Data Injection
    • Supporting the VAST Survey Tag
    • Firing the Error Tag
    • Special Ad Tag Test Cases
    • Custom Ad Delivery Options
  • Visual Elements During Playback
    • Configuring the Player Display
    • Ad and Click-Through Notices
    • Skip Linear Ad Options
  • HTML5 and Javascript Support
    • The Javascript API
  • Ad Metrics
    • Tracking with Google Analytics
  • General Test Cases
    • Specifying a Start Time (ad or show stream)
    • Supporting streams and ads without a duration specified
