OVA for JW Player 4 - Worked Examples

A number of examples have been included with this release to showcase the available range of configuration options. All of these examples require JW Player 4.6 or greater.

Broadly speaking, the examples have been organised into the following groupings:

  • Illustrating the Supported Ad Formats
  • Delivering RTMP Linear Ads
  • Minimal Setup
  • Deferred Ad Loading
  • Configuring the Ad Notice
  • Skip Linear Ad Options
  • Configuring the Ad Servers
  • Supporting Companions
  • Custom Ad Delivery Options
  • The Javascript API
  • JW Player Specific Configuration Options
  • Supporting streams and ads without a duration specified
  • Configuring Overlays
  • Special Cases
  • Specifying a Start Time (ad or show stream)
  • Direct VAST Data Injection
  • VAST Wrapper

A number of the examples use an OpenX installation that we are running at Open Video Ads. To gain more insight into the zone/banner setup that we have in place for the examples, please read this short guide.
