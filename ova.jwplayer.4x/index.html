OVA for JW Player 4 - Version 1.0.0

Welcome to the V1.0.0 release of the Open Video Ads streamer for Longtail Video's JW Player 4.

This package contains the binary (SWF) files required to deploy the plugin with JW Player 4.6 (or greater) from Longtail Video.

Several examples are also provided to help illustrate the various configurations for the plugin.

Further documentation can be found on the detailed plugin documentation pages at the google code repository www.openvideoads.org
