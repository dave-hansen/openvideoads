1. Introduction

OVA supports the ability to configure a button on the player display to allow the viewer to skip a linear non-interactive ad stream.

This button can be displayed as a default image, custom image or custom HTML text.

It can also be enabled to only be displayed if the ad is of a certain duration.

2. Working Example

The following example illustrates the Skip Ad button in it's default state. The appearance and wording of the button may be changed via OVA configuration options.

{ "ads": { "skipAd": { "enabled": true }, "schedule": [ { "position": "pre-roll", "tag": "OVA_AD_TAG_LINEAR_STREAM_WITH_80x300_COMPANION" } ] }, "debug": { "levels": "fatal, config, vast_template" } }
{ "playlist": [ { "url": "OVA_HTTP_SHOW_STREAM" } ], "plugins": { "controls": { "autoHide": "never" }, "ova": { "url": "OVA_FLOWPLAYER_PLUGIN_SWF" } } }
{ "flashplayer": "OVA_JW5_PLAYER_SWF", "playlist": [ { "file": "OVA_HTTP_SHOW_STREAM" } ], "width": 450, "height": 300, "controlbar": { "position": "bottom" }, "plugins": { } }

3. Enabling the button

A default Skip Ad button can be enabled as follows:

{
     "ads": {
          "controls": {
               "skipAd": {
                   "enabled": true
               }
          },
          ....
     }
}

3.1 The default button

By default a button with a pre-defined image is used as illustrated by the worked example above.

That image can be overridden.

3.2 Using a Custom Image for the Skip Ad Button

To override the standard skip ad button image, use the image property and specify the new image width and height:

{
    "ads": {
         "controls": {
                "skipAd": {
                     "enabled": true,
                     "image": "../../images/my-new-skip-button.jpg",
                     "width": 65,
                     "height": 15
                }
        },
        ....
    }
}

3.3 Using Region Attributes with the Skip Ad Buttons

It is also possible to use the region attributions to declare a skip button - background color, margins, borders and html content can be specified to construct the look of the button.

{
     "ads": {
           "controls": {
                "skipAd": {
                     "enabled": true,
                     "html": "<p>SKIP!</p>",
                     "region": {
                            "id": "my-new-skip-ad-button",
                            "verticalAlign": 3,
                            "horizontalAlign": 3,
                            "backgroundColor": "#FF3300",
                            "opacity": 0.8,
                            "borderRadius": 15,
                            "padding": "0 1 1 13",
                            "width": 60,
                            "height": 20
                     }
                }
           },
           ....
      }
}

4. Showing the Skip Button on Ads with a minimim duration

It is possible to configure the Skip Ad button so that it only shows on ads that are longer than a specific duration.

For example, the following configuration configures the Skip Ad button to show only on ads that are 10 seconds or longer in duration.

{
     "ads": {
          "controls": {
               "skipAd": {
                   "enabled": true,
                   "minimumAdDuration": 10
               }
          },
          ....
     }
}

5. Showing after a number of seconds

To ensure that the skip ad button only shows after a nominated number of seconds, use the "showAfterSeconds" option as follows:

{
    ....
    "ads": {
         "skipAd": {
              "enabled": true,
              "showAfterSeconds": 10
         },

6. Showing for a number of seconds

To ensure that the skip ad button is displayed for a limited number of seconds, use the "showForSeconds" option as follows:

{
    ....
    "ads": {
         "skipAd": {
              "enabled": true,
              "showForSeconds": 5
         },

If you wish to show the skip ad button after a nominated number of seconds for a specific number of seconds, use the following config:

{
    ....
    "ads": {
         "skipAd": {
              "enabled": true,
              "showAfterSeconds": 10,
              "showForSeconds": 5
         },
