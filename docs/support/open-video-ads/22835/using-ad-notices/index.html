1. Introduction

OVA provides a standard ad notice that is displayed when a linear ad is played.

2. The standard ad notice

The standard notice worded as "This advertisement runs for X seconds" is illustrated below.

{ "ads": { "schedule": [ { "position": "pre-roll", "tag": "OVA_AD_TAG_LINEAR_STREAM_WITH_80x300_COMPANION" } ] }, "debug": { "levels": "fatal, config, vast_template" } }
{ "playlist": [ { "url": "OVA_HTTP_SHOW_STREAM" } ], "plugins": { "controls": { "autoHide": "never" }, "ova": { "url": "OVA_FLOWPLAYER_PLUGIN_SWF" } } }
{ "flashplayer": "OVA_JW5_PLAYER_SWF", "playlist": [ { "file": "OVA_HTTP_SHOW_STREAM" } ], "width": 450, "height": 300, "controlbar": { "position": "bottom" }, "plugins": { } }

The wording, format, positioning and appearance of this notice may be modified via OVA configuration options.

3. Modifying the notice wording

The notice configuration option exists to support the modification of the standard ad notice.

The following code snippet illustrates how to modify the ad notice wording for a scheduled pre-roll linear ad.

{
    ...
          "ads": {
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag",
                      "notice": {
                            "message": "<p class="smalltext" align="right">Publicité -
                            Votre vidéo dans _countdown_ seconde(s)</p>"
                      }
                  }
              ]
         }
   ...
}

It is possible to modify the ad notice at either an ad slot or general level. The following code snippet illustrates how to modify the ad notice at the general level:

{
    ...
          "ads": {
              "notice": {
                      "message": "<p class="smalltext" align="right">Publicité - 
                      Votre vidéo dans _countdown_ seconde(s)</p>"
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}

Ad notices modified at the general level are applied across all ad slots.

Note: Where the ad notice wording is being configured via an XML based configuration file (e.g. in the case of a JW Player external configuration file), care must be taken to ensure that the HTML tags included in the "message" configuration do not break the XML structure. To ensure that the HTML tags do not break the XML structure, encode them as follows:

{
    ...
          "ads": {
              "notice": {
                   "message": "&lt;p class="smalltext" align="right"&gt;My New Ad Notice - _seconds_ seconds&lt;/p&gt;"
              },
              ...

"&lt;" should be used in place of the "<" character and "&gt;" should replace all instances of the ">" character in the HTML tags.

4. Modifying the notice appearance

4.1 Using small text

OVA provides a pre-defined text style that allows the ad notice to be displayed with small text.

The following code snippet illustrates how to force the ad notice to use that small text styling:

{
    ...
          "ads": {
              "notice": {
                     "textStyle": "smalltext"
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}

You can see this option in action here.

4.2 Modifying the display region

It is possible to modify the positioning of the ad notice and the styling of the ad notice region completely using a custom region declaration.

The following code snippet illustrates how to do this.

{
    ...
          "overlays": {
              "regions": [
                  {
                      "id": "my-ad-notice",
                      "verticalAlign": "bottom",
                      "horizontalAlign": "right",
                      "backgroundColor": "transparent",
                      "width": "100pct",
                      "height": 20,
                      "style": ".smalltext { font-style: italic; font-size:10; }"
                  }
              ]
          },
          "ads": {
              "notice": {
                     "region": "my-ad-notice",
                     "text": "Only 20 seconds to go until your programme begins" 
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}

You can see this option in action here.

5. Deploying a count-down notice

OVA provides a _countdown_ keyword that can be placed in the ad notice text. This keyword allows OVA to display the remaining time of an ad as part of the ad notice during playback.

The following code snippet illustrates how to define a new ad notice message using this keyword.

{
    ...
          "ads": {
              "notice": {
                     "text": "Only _countdown_ seconds to go until your programme begins"
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}

Additionally, if you are happy with the standard OVA ad countdown notice "Your advertisement runs for _countdown_ seconds", you can instruct OVA to use the countdown notice via the type configuration option. The following code snippet illustrates how to do this.

{
    ...
          "ads": {
              "notice": {
                     "type": "countdown"
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}

You can see this option in action here.

6. Turning off the notice

The ad notice can be turned off complete via the show option as illustrated below.

{
    ...
          "ads": {
              "notice": 
                     "show": false
              },
              "schedule": [
                  {
                      "position": "pre-roll",
                      "tag": "http://my-ad-provider/ad-tag"
                  }
              ]
         }
   ...
}
