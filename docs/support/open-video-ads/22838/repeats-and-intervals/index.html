1. Introductions

OVA offers a repeat" option to allow ad slot definitions to be duplicated.

2. Using "repeat"

The following example illustrates the repeat option in action - three pre-rolls are played before the show stream.

{ "ads": { "servers": [ { "type": "OpenX", "apiAddress": "http://openx.openvideoads.org/openx/www/delivery/fc.php", "allowAdRepetition": true } ], "schedule": [ { "zone": "5", "position": "pre-roll", "repeat": 3 } ] }, "debug": { "levels": "fatal, config, vast_template, playlist" } }

To specify that a specific ad slot is to be repeated a number of times, add the repeat option to the ad slot declaration along with a number representing the times the ad slot is to be duplicated.

Note: At this time, OVA for Flowplayer mid-roll positions cannot be repeated without a spacing of at least 2 seconds between each mid-roll. This means that it is mandatory to use the "interval" option with OVA for Flowplayer mid-roll ad slots that use the "repeat" option.

3. Spacing Repeated Ad Slots

It is possible to schedule repeated mid-stream (overlay and mid-roll) ad slots so that the start times are automatically spaced by a specific number of seconds.

The interval ad slot option has been provided to achieve this.

The option should be declared as follows:

{ 
   ...
   "ads": {
        "schedule": [
             {
                 "zone": "5",
                 "position": "mid-roll",
                 "startTime": "00:00:10",
                 "repeat": 3,
                 "interval": 10
             }
        ]
   }
   ...
}

The interval value should always be specified as the number of seconds that is to be placed between each repeating instance of the ad slot.
