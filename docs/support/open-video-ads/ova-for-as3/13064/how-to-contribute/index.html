How to Contribute

Interested in helping? Fantastic. Open Video Ads relies on the support of volunteers all over the world.

You can help with the development of the OVA regardless of whether or not you are a software engineer.

Here are some of the ways you can contribute....

Requesting New Features/Supporting Additional Ad Servers

The OVA project maintains an issues register and release roadmap on the developer site (developer.longtailvideo.com/ova). If you would like to a feature request, email the request to enquiries@openvideoads.org and it will be assessed and entered into the OVA register and release roadmap.

Every week additional "out-of-the-box" support is added for new VAST compliant ad servers. If you have added support to the framework for a new ad server and wish to contribute that code back into the OVA project, email enquiries@openvideoads.org to request write access to the SVN repository.

Participate in the Forums

The OVA project runs a number of forums. Help users use the OVA framework or OVA plugin products by contributing to the discussions in those forums.

Help Produce Documentation

We are always working to improve our documentation. It's a time consuming and often frustrating process documenting the code that we write, but we try our best to produce more comprehensive documentation with each new release. If you are using the framework and have documented an aspect of it and would like to contribute that documentation back into this community site, email enquiries@openvideoads.org.

Testing

The OVA framework and player plugins are constantly going through certification by Ad Networks deploying OVA as part of their overall video advertising solution. If you are certifying an OVA product for your organisation and have developed a set of test cases that you would like to contribute to the OVA community, email enquiries@openvideoads.org.

Reporting Bugs

We're not perfect and often uses of the OVA products report bugs that they find as they test or deploy the technology. If you find a bug and would like to log it in our issues system, email enquiries@openvideoads.org. All bugs are recorded in the issues log on the OVA developer site.

Fixing Bugs

Have you fixed a defect in the codebase. To integrate that fix back into the next OVA release, email the details of the fix and a diff to enquiries@openvideoads.org. 

Show Your Support

If you enjoy Open Video Ads or find it to be useful, put a link to Open Video Ads on your site. You can do this with a text link, one of the following OVA icons, or link to www.openvideoads.org. There is no better way to support us than to show you care.
