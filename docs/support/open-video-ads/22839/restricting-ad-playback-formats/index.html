Restricting Ad Playback Formats

Purpose

This short guide explores how to use the "mime-type" exclusion rules of OVA to limit the possible set of ad types that are played.

1. Introduction

A VAST response often includes ad streams of differing bitrates and mime-types.

Sometimes a player instance will only require ad types of specific mime types - for example where the player is being used to deliver an audio-only experience. In that case, only MP3 (or other audio format) based ads are acceptable for playback.

The mime-type exclusion rules supported by OVA allow the set of acceptable ad formats to be restricted to a defined list.

2. Example - Delivering Audio Ads

The following example illustrates how to use the mime-type exclusion rules to deliver audio only linear ad experience.

{ "providers": { "http": "PLAYER_SPECIFIC_AUDIO_PROVIDER" }, "ads": { "acceptedLinearAdMimeTypes": [ "audio/x-mpeg-3" ], "filterOnLinearAdMimeTypes": true, "notice": { "show": false }, "schedule": [ { "position": "pre-roll", "tag": "OVA_AUDIO_AD_TAG" } ] }, "debug": { "levels": "fatal, config, vast_template, http_calls" } }
{ "playlist": [ { "url": "OVA_AUDIO_SHOW_STREAM" } ], "plugins": { "controls": { "fullscreen": false, "autoHide": false, "height": 30 }, "audio": { "url": "OVA_FLOWPLAYER_AUDIO_PLUGIN_SWF" }, "ova": { "url": "OVA_FLOWPLAYER_PLUGIN_SWF" } } }
{ "flashplayer": "OVA_JW5_PLAYER_SWF", "playlist": [ { "file": "OVA_AUDIO_SHOW_STREAM" } ], "width": 450, "height": 24, "controlbar": { "position": "bottom" }, "plugins": { } }

3. The Configuration Options

Two configuration options have been added to the "ads" configuration block to allow media files to be excluded during OVA search to identify relevant linear streams to playback.

The two options are illustrated below.

"ads": {
      "acceptedLinearAdMimeTypes": [ "video/x-flv" ],
      "filterOnLinearAdMimeTypes": true,
      ...
}

In this example, only FLV based ad streams will be selected.

acceptedLinearAdMimeTypes is an Array based configuration property. A set of mime types can be configured as per a normal Array based definition. For instance, to limit the filtering to FLV and MP4, the following declaration should be made:

"acceptedLinearAdMimeTypes": [ "video/x-flv", "video/x-mp4"]

By default:

  • In the AS3 framework filterOnLinearAdMimeTypes is false
  • In the OVA JW4, JW5 and Flowplayer plugins, filterOnLinearAdMimeTypes is true and acceptedLinearAdMimeTypes is set to ["video/x-flv","video/x-mp4","application/x-shockwave-flash"]

To turn off the default filtering in the OVA plugins just set filterOnLinearAdMimeTypes to false - all media files elements of the VAST response will then be searched for a match to playback.
